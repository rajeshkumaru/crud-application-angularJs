## General info
This project is simple Crud functionality.
	
## Technologies
Project is created with:

* Angularjs: 1.5+
* Es6
	
## Setup
To run this project, install it locally using npm:

$ cd ../My_App
$ npm install
$ npm run start / npm run server

For test
$ npm run test

For build
$ npm run build

## output
Hit the URL into your browser - http://localhost:8080/
